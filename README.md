  # Baroque — Fashion Store Template

  A static, multi-page storefront template for a clothing/fashion brand. Pure
  HTML/CSS — no build step, no backend. Open any page in a browser.
  
  ## Pages

  | File | Purpose |
  |------|---------|
  | `Home.html` | Landing page / shopfront |
  | `Celebrity Style.html` | Celebrity-style product category |
  | `Unstitched.html` | Unstitched product category |
  | `Add to Cart.html` | Cart view |
  | `Checkout.html` | Checkout / order summary |

  ## Usage

  Clone and open `Home.html` directly, or serve the folder with any static server:

  ```bash
  python -m http.server 8000
  # then open http://localhost:8000/Home.html
